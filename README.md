# nest-boom

![](https://img.shields.io/badge/language-Node-yellow) ![](https://img.shields.io/badge/version-0.6.4-brightgreen) [![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)]()

nest-cli generate resource 명령을 커스텀한 도구입니다.
nest.ts + sequelize-typescript 기반입니다.

이렇게 깔고

```
npm i -g nest-boom
```

이렇게 쓰면 됩니다.

```
nest-boom fire-ball
```
